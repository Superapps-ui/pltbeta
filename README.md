# pltBeta/full

Программа просто подставляет никнейм в ссылки на соцсети и открывает их в браузере. Никакого взлома, сбора данных или чего-то опасного.

# Установка в Termux

```bash
pkg update && pkg upgrade -y
pkg install python git -y
pip install colorama
git clone https://github.com/tvoy_nick/pltfull
cd pltfull
python pltfull.py
```

# Как работает

Ты вводишь ник → программа открывает ссылки типа:

· https://t.me/ник
· https://github.com/ник
· https://reddit.com/user/ник

Всё. Больше ничего.
